Server Rules

To ensure everyone has a great experience, please follow these rules:

Rule Description
Be Respectful No harassment, bullying, or hate speech. Treat others as you'd like to be treated. We're here to have fun together.
No Griefing Don't destroy or steal from others' builds without permission. Respect the creations of fellow players.
No Cheating No hacks, exploits, or unfair advantages. Play fair! All violations result in permanent bans.
Keep It Family-Friendly No inappropriate content, language, or builds. This server is for players of all ages.
Report Issues If you see something wrong, report to admins via Discord or in-game. We can't fix what we don't know about.
Have Fun! Enjoy the server, make friends, and create amazing memories. That's what gaming is all about!
